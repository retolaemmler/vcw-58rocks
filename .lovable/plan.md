

# Vibe Coding Workshop — Landing Page

A sleek, modern single-page landing site for a full-day Lovable.dev bootcamp in Switzerland. Light/white theme with vibrant teal, purple, and blue gradient accents. Fully responsive, mobile-first design.

## Design & UX
- Light/white background alternating with light grey (#f8f9fa) sections
- Teal/turquoise primary accent, purple & blue gradient secondary accents
- Soft shadows, subtle card borders, lots of whitespace
- Smooth scroll navigation, scroll-triggered animations
- Modern sans-serif typography (Inter / Space Grotesk)
- Subtle geometric patterns and gradient blobs as background decoration
- Fully responsive mobile-first layout
